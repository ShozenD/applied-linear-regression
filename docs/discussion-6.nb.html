(a) Plot the data

We can produce a simple plot by calling plot() on cats.

plot(cats)

To plot multiple regression lines based on categorical variables, we can pass the Sex variable to color in the aesthetics of the ggplot function.

ggplot(cats, aes(y = Bwt, x = Hwt, colour=Sex, shape=Sex)) + geom_point() + geom_smooth(method="lm", fill = NA)

From the plot we can see that the heart weight and body weight regression line for female cats have a slightly smaller intercept and slope when compared to that of male cats. Also the range for heart weight and body weight is also smaller.

(b) Test the assumption \(\varepsilon_{i} ~ Normal(0,\sigma_{\varepsilon}^2)\)

We can test for normality of errors by fitting the linear model and conducting a Shapiro-Wilk normality test

lm.fit = lm(Bwt~., data = cats)
shapiro.test(lm.fit$residuals)

    Shapiro-Wilk normality test

data:  lm.fit$residuals
W = 0.99258, p-value = 0.6597

Because the p-value is much greater than 0.1 we cannot reject the null hypothesis and thus conclude that the distribution of errors is normal.

Scheffes method

p = 3
g = 5
n = dim(cats)[1]
X = model.matrix(lm.fit)
middle.part = solve(t(X)%*%X)
X.new = cbind(1, c(1, 1, 1, 0, 0), 1:5)
sigma.hat = summary(lm.fit)$sigma
width.half = sqrt(g*qf(0.95, g, n-p)*(1+apply(X.new, 1, function(x){t(x)%*%middle.part%*%x})))*sigma.hat
Schef.pred = matrix(rep(predict(lm.fit, data.frame(Sex = c('M','M','M','F','F'), Hwt = 1:5)), 2), ncol = 2)
Schef.pred[, 1] = Schef.pred[, 1] - width.half
Schef.pred[, 2] = Schef.pred[, 2] + width.half
colnames(Schef.pred) = c('lwr', 'upr')
print(Schef.pred)
